# EventExplorer React Native (Expo)

A minimal React Native app (Expo) that displays event entries.

Quick start

1. Install dependencies:

```bash
cd EventExplorerApp
npm install
```

2. Run the app with Expo:

```bash
npx expo start
# then press 'a' to open on Android emulator, or scan the QR with Expo Go on a device
```

Notes

- This project uses Expo for simplicity and quick Android testing.
- If you want a native Android build, run `eas build -p android` after configuring `eas.json` and an Expo account.
